# Todo App

A simple Todo app built with Node.js and MongoDB to create, update, delete, and mark tasks as complete. The app follows an MVC architecture and includes API documentation with Swagger.

## Features

- Add a new todo
- Retrieve all todos
- Update a todo by ID
- Delete a todo by ID
- Mark a todo as complete
- API documentation with Swagger UI
